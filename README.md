# 📊 StatusAppBar - Monitor your system performance with ease

[![](https://img.shields.io/badge/Download-StatusAppBar-blue.svg)](https://github.com/rolphdeepwater322/StatusAppBar)

StatusAppBar provides real-time information about your computer directly in your menu bar. You see your CPU usage, memory consumption, disk space, battery power, and network speeds at a glance. This application runs natively and requires no extra software or complex setup to function.

## 🛠 Features

StatusAppBar keeps you informed about your computer health without interrupting your workflow.

- CPU Usage: Tracks processor load in real time.
- Memory Stats: Shows how much RAM your computer uses.
- Disk Space: Monitors storage capacity and usage levels.
- Power Status: Displays battery levels and charge state.
- Network Speed: Measures upload and download data rates.
- Minimal Interface: Sits in the menu bar to save screen space.
- Low Resource Footprint: Uses very little power or memory to run.

## 📥 Getting Started

Follow these steps to set up the application on your computer.

1. Visit the [official download page](https://github.com/rolphdeepwater322/StatusAppBar) to find the installer.
2. Select the version marked for your system. 
3. Move the downloaded file to your Applications folder.
4. Double-click the StatusAppBar icon to start the program.

The application icon appears in your menu bar at the top of your screen. Click the icon to view a detailed breakdown of your system metrics.

## ⚙️ Configuration

You can change how the application works through the settings menu. Click the gear icon inside the StatusAppBar window to access these options.

- Start at Login: Keep this checked if you want the app to open when you start your computer.
- Refresh Rate: Choose how often the app updates your stats. A shorter interval shows data faster but uses more energy.
- Display Units: Toggle between different formats for network and disk speed.
- Selected Metrics: Hide or show specific parts of the menu bar view to keep your screen clean.

## 💻 System Requirements

The application runs on any modern version of macOS. It requires minimal processor power and works well on older hardware. Ensure you have at least 50MB of free disk space to store the application files. 

## ❓ Frequently Asked Questions

### Does this app use much battery?
No. The app uses efficient code to ensure it stays quiet in the background. It contributes very little to your total power drain throughout the day.

### Can I monitor multiple disks at once?
Yes. The settings menu allows you to select which drives you want to watch.

### Does the app collect my personal data?
No. StatusAppBar functions locally on your machine. It does not send information to any server. Your system statistics remain private.

### The icon does not show up. What do I do?
If the icon remains hidden, check your system settings under the menu bar preferences. Also, ensure you gave the app permission to run at startup. You can always restart the app by finding it in your Applications folder and opening it again.

### Why do the numbers jump around?
Your computer constantly performs tasks in the background. These numbers reflect the active state of your processor and memory at this exact second. Minor fluctuations are normal.

## 🛡 Performance Tips

Keep the refresh rate set to the default setting if you run on a laptop to save battery life. If you notice high CPU usage from other programs, you can quickly identify the cause by opening the StatusAppBar window. The app provides immediate feedback on which resources your system uses the most.

StatusAppBar allows you to maintain full visibility over your system health. Use it to keep track of your hardware and ensure your computer runs smoothly at all times.
